日常编写公式用到太多各类导数符号了！必须要总结在这里：

**（1）偏导符号：\\partial x**

```
$ \frac{\partial f}{\partial x} $  # 一阶$ \frac{\partial ^{n} f}{\partial x^{n}} $  # n阶
```

**（2）求导符号：\\mathrm{d} x**

```
$ \frac{\mathrm{d} y }{\mathrm{d} x} $  $ \frac{\mathrm{d}^{n} y }{\mathrm{d} x^{n}} $  
```
